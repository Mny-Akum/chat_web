## 功能:cyclone:

1. ​群组聊天
2. 私人聊天
3. 邮箱注册
4. 精美的注册登录页面
5. 精心制作的用户列表
6. 仿QQ聊天

   1. 仿QQ聊天气泡

   2. 仿QQ聊天时间显示
7. 精心制作且炫酷的输入框
8. 精心制作且炫酷的消息提示
9. 精心制作的个人主页
10. 用户是否在线显示
11. 自定义聊天头像(支持动图)
12. 悬浮可拖动菜单

## 运行:zap:

- 确保您的电脑安装了node.js 经试验此服务可以在v14.18.2.版本下运行

- 确保你的电脑配置好了node.js的环境

*下载此项目后通过```npm run serve```*命令运行

*填写ip之后请注册账号，然后登录使用*
